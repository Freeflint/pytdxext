# Pytdxext 
是基于pytdx1.7.2以及Mootdx0.8.10的源码扩展(由于pytdx作者已经不再维护,本项目主要目的是持续维护pytdx源码),考虑到mootdx对pytdx的二次封装更好用,所以本项目的核心入口接口全部使用mootdx,而pytdx的核心接口将不直接开放出来,在此非常感谢以上两个项目作者的辛勤付出。
（注意此项目为仅本人自用项目）
