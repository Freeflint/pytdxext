# 安装方式

## 普通安装

```shell
pip install pytdxext
```

## 升级版本

```shell
pip install -U pytdxext
```

## 源码安装

直接远程源码安装

```shell
pip install git+https://github.com/pytdxext/pytdxext.git
```

或下载后手动安装

```shell
git clone https://github.com/pytdxext/pytdxext.git

cd pytdxext

python setup.py install
```
